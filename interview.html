Top 50 JavaScript interview question


1.What is javascript? Feature of javascript?

Ans. JavaScript is a popular programming language used for creating interactive and dynamic web pages. It is a client-side language, meaning it runs on the user's computer rather than on the server. JavaScript allows web developers to add interactivity, create animations, and validate user input on a webpage.


2.Is javascript is strictly typed and dynamic typed? Explain with example

Ans. JavaScript is a dynamically typed language, which means that variables do not have a specific type assigned to them. For example, a variable can be assigned a string value and then later assigned a numerical value without any issues. However, JavaScript does have some type checking that occurs at runtime.


3.Why do we need javaScript? explain?

Ans. We need JavaScript for several reasons, including: Adding interactivity and dynamic content to web pages
Validating user input and handling form submissions
Creating animations and special effects
Building web-based games and applications
Communicating with servers to retrieve or send data
Creating mobile applications using frameworks such as React Native.


4.Types of JavaScript?

Ans. Types of JavaScript include: Vanilla JavaScript: This refers to plain JavaScript without the use of any libraries or frameworks.
Node.js: This is a runtime environment that allows JavaScript to be run on servers.
React: This is a JavaScript library used for building user interfaces.
Angular: This is a JavaScript framework used for building web applications.
Vue.js: This is a JavaScript framework used for building user interfaces.


5.What are the variables? types of variable? Difference between them explain.

Ans. Variables are containers that hold values in a program. There are three types of variables in JavaScript: var: This keyword is used to declare a variable that has function scope. It can be redeclared and updated throughout the program. let: This keyword is used to declare a variable that has block scope. It can be updated but not redeclared within the same block. const: This keyword is used to declare a variable that has block scope and cannot be updated or redeclared.


6.What is type of operator?

Ans.The typeof operator is used to determine the data type of a value or variable in JavaScript.


7.Difference between == & ===.

Ans. The double equals (==) operator compares the values of two operands for equality, while the triple equals (===) operator compares the values and types of two operands for equality.


8.What is an array in JavaScript? explain few array methods.

Ans. An array is a collection of values that are stored in a single variable. Some common array methods in JavaScript include: push(): Adds one or more elements to the end of an array.
pop(): Removes the last element from an array.
shift(): Removes the first element from an array.
unshift(): Adds one or more elements to the beginning of an array.
slice(): Returns a new array that contains a portion of an existing array.
splice(): Adds or removes elements from an array.


9.Difference between slice and splice? explain with syntax.

Ans. The slice() method returns a new array that contains a portion of an existing array without modifying the original array. The splice() method adds or removes elements from an array and modifies the original array. The syntax for slice() is:
array.slice(startIndex, endIndex);
The syntax for splice() is:
array.splice(startIndex, countToRemove, item1, item2, ...);


10.What is variable hoisting?

Ans. Variable hoisting is a behavior in JavaScript where variable declarations are moved to the top of their respective scope, regardless of where they are declared within the scope. This allows variables to be used before they are declared.


11.Create an object, read an opbject, edit an object and delete an object?

Ans. To create an object in JavaScript, we can use object literal notation:
const myObject = {
name: "John",
age: 30,
city: "New York"
};
To read a property from an object, we can use dot notation:
console.log(myObject.name);
To edit a property in an object, we can use dot notation:

12.Explain object method?

Ans. Object methods are functions that are defined inside an object and can be called on the object. They allow us to perform operations on the object's data or properties. For example, we can define a method called "calculateArea" on an object representing a rectangle that calculates the area of the rectangle based on its width and height.


13.Create opbject using constructor method?

Ans.An object can also be created using the constructor method in JavaScript. This involves defining a constructor function that creates a new object with certain properties and methods. For example:


function Person(name, age) {
this.name = name;
this.age = age;
this.greet = function() {
console.log(`Hello, my name is ${this.name} and I am ${this.age} years old.`);
};
}

const john = new Person("John", 30);
john.greet(); // Output: Hello, my name is John and I am 30 years old.

14.What is 'this' keyword used for? Explain with example.

Ans.The "this" keyword is used to refer to the current object or instance of an object in JavaScript. It is often used inside methods to refer to the object that the method is called on. For example:


const person = {
name: "John",
age: 30,
greet: function() {
console.log(`Hello, my name is ${this.name} and I am ${this.age} years old.`);
}
};
person.greet(); // Output: Hello, my name is John and I am 30 years old.


15.What is local variable and global variables? Explain with example.

In JavaScript, a local variable is a variable that is declared inside a function and can only be accessed within that function. A global variable is a variable that is declared outside of a function and can be accessed from anywhere in the program. For example:


function sayHello() {
const message = "Hello";
console.log(message); // Output: Hello
}

sayHello();

function greet() {

16.What is function? Difference types function.

A function is a block of code that performs a specific task. There are several types of functions in JavaScript, including:
Function declarations: These are defined using the "function" keyword followed by a name and a block of code.
Function expressions: These are defined by assigning a function to a variable or constant.
Arrow functions: These are a shorthand way of defining a function using the "=>" syntax.

17.What is spread operator amd rest operator? Write an example.

The spread operator (...) and rest operator (...) are two related operators in JavaScript. The spread operator can be used to expand an array or object into individual elements, while the rest operator can be used to collect multiple arguments into an array. For example:


const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4, 5]; // Output: [1, 2, 3, 4, 5]

function sum(...args) {
return args.reduce((total, num) => total + num);
}

console.log(sum(1, 2, 3, 4)); // Output: 10


18.What is closure?Write program for getting closure.

A closure is a function that has access to variables in its outer function scope, even after the outer function has returned. Closures are created by defining a function inside another function and returning it. For example:


function outer() {
const message = "Hello";
function inner() {
console.log(message);
}
return inner;
}
outer();


or, A closure is a function that has access to its parent's variables, even after the parent function has been executed. It is created when a function is defined inside another function and the inner function refers to variables in the outer function. The inner function can access the variables of the outer function, even after the outer function has returned. Here's an example:


function add(x) {
return function(y) {
return x + y;
}
}
let add5 = add(5);
console.log(add5(3)); // Output: 8
console.log(add5(5)); // Output: 10


19.What is callback function? Write a program.

Ans. Callback Function:
A callback function is a function that is passed as an argument to another function and is executed when the first function is done. Callback functions are often used in asynchronous programming, where a function is called and then another function is called when the first function is finished. Here's an example:


function multiply(x, y, callback) {
let result = x * y;
callback(result);
}

function displayResult(result) {
console.log("The result is: " + result);
}

multiply(5, 3, displayResult); // Output: The result is: 15


20.Difference between map and forEach? Explain with example.

Difference between map() and forEach():
Both map() and forEach() are methods that can be used to iterate over an array in JavaScript, but they differ in their return value and the way they modify the original array.
map() creates a new array with the results of calling a provided function on every element in the original array. The original array is not modified. Here's an example:


let numbers = [1, 2, 3, 4];

let doubled = numbers.map(function(num) {
return num * 2;
});

console.log(doubled); // Output: [2, 4, 6, 8]
console.log(numbers); // Output: [1, 2, 3, 4]


forEach() executes a provided function once for each array element. It does not return a new array. The original array is not modified. Here's an example:


let numbers = [1, 2, 3, 4];

numbers.forEach(function(num) {
console.log(num);
});

// Output:
// 1
// 2
// 3
// 4

console.log(numbers); // Output: [1, 2, 3, 4]

22.What is NaN property in JavaScript?

Ans. NaN property represents the “Not-a-Number” value. It indicates a value that is not a legal number.

typeof of NaN will return a Number.

To check if a value is NaN, we use the isNaN() function,
Note- isNaN() function converts the given value to a Number type, and then equates to NaN.
isNaN("Hello") // Returns true
isNaN(345) // Returns false
isNaN('1') // Returns false, since '1' is converted to Number type which results in 0 ( a number)
isNaN(true) // Returns false, since true converted to Number type results in 1 ( a number)
isNaN(false) // Returns false
isNaN(undefined) // Returns true
